## 下载gnome-tweak工具
```bash
sudo yum install gnome-tweak-tool #Fedora 
sudo apt install gnome-tweak-tool #Debian
```
<!--more-->
然后按Alt + F2打开命令窗口，输入gnome-tweaks（Fedora）/ gnome-tweak-tool（Debian），或者在应用里选择优化，在更多的窗口标题栏中，将最大化，最小化打开即可。

转换单元格格式：使用设置标题快捷键，强制转换 cell 格式到 markdown       
设置 markdown 标题快捷键    
```shell
1	# 设定 1 级标题	仅在 markdown 状态下时建议使用标题相关快捷键，如果单元处于其他状态，则会强制切换到 markdown 状态
2	# 设定 2 级标题
3	# 设定 3 级标题
4	# 设定 4 级标题
5	# 设定 5 级标题
6	# 设定 6 级标题
```

删除单元格：`esc` -- 双击按键 `d`  

插入单元格    
```shell
A	# 在上方插入新单元
B	# 在下方插入新单元
```
开关行号 `L	# 开关行号`
